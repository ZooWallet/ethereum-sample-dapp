# Ethereum Sample Dapp

It's a sample Dapp that shows you how to use the MetaMask wallet and Dapp browsers (e.g. [Dapp Pocket](https://dapppocket.io/)).

## How to use?

1. clone this repo
2. run `npm i`
3. run `npm run start` to start the app at http://localhost:3000
